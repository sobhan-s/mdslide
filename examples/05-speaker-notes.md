## Keyboard Controls Cheat Sheet

Use these keys when presenting in your browser:

| Key           | Control        | Description                 |
| :------------ | :------------- | :-------------------------- |
| `Space` / `→` | Next           | Go to the next slide/bullet |
| `←`           | Previous       | Go back one slide/item      |
| `f` / `F`     | Fullscreen     | Toggle fullscreen mode      |
| `p` / `P`     | Presenter View | Open synced presenter view  |

Briefly review the shortcut table during delivery. Emphasize full-screen usage.
